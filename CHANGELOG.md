# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0-alpha.1+1] - 2026-03-30

### Added

#### AI 管理模块
- AI 代理管理功能
- 浏览器配置管理
- Ollama 模型集成和 GPU 监控 (10 个端点)

#### 系统设置
- 仪表盘备忘录功能
- Passkey 管理功能
- 系统配置和快照管理 (15 个端点)

#### 工具箱
- ClamAV 病毒扫描管理
- Fail2ban 入侵防护管理
- FTP 管理服务

#### 运行时管理
- PHP 扩展管理
- Node 模块管理
- Supervisor 进程管理
- 完整运行环境管理 (24 个端点)

#### 网站管理
- 批量操作功能
- SSL 账户管理
- 完整的网站、域名、SSL 和代理管理 (65 个端点)

#### 容器管理
- 增强容器功能和 UI 优化
- 容器卡片和统计视图优化
- 完整 Docker 容器和镜像管理 (50+ 个端点)

#### 文件管理
- 回收站功能
- 传输管理器
- 全面文件操作和传输功能 (28 个端点)

#### 服务器管理
- 添加删除功能
- 分组中心
- 完整的主机监控和系统管理 (18 个端点)

#### 数据库管理
- 完整的数据库操作 (34 个端点)
- 支持 MySQL/MariaDB、PostgreSQL、Redis

#### 备份管理
- 完整的备份操作和恢复功能 (14 个端点)

#### 其他新增
- 完整的 425+ API 端点覆盖 (26 个 V2 API 模块)
- 31 个全面的数据模型文件
- 三轮 API 验证，生产就绪状态确认

### Changed

#### UI 优化
- 优化容器卡片和统计视图 UI 设计
- 优化服务器详情页面和路由管理
- 优化容器模块功能并优化界面布局
- 更新应用图标和相关资源

#### 系统优化
- 优化系统设置模块的可用性检查与更新逻辑
- 优化代码结构和可读性

#### 依赖更新
- 更新依赖库到最新版本
- 更新 API 覆盖度文档

### Fixed

#### Bug 修复
- 修复命令模块 API 契约偏差
- 修复文件预览和上传历史页面的大小格式化

#### 其他修复
- 各种性能优化和稳定性改进
