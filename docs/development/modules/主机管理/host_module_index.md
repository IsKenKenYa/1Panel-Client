# 主机管理模块索引

## 模块定位

主机管理模块在 Phase 1 中拆成主机资产、SSH、进程三段推进。Week 1 已完成主机资产 API 口径澄清与分组底座接入准备，Week 2 落地主机资产主链路。

### 核心职责

1. **主机接入** - 添加、编辑、删除主机
2. **连接验证** - 按信息测试与按 ID 测试
3. **分组管理** - 统一接入共享 group 底座
4. **树形选择** - 为后续多模块复用提供 host tree

