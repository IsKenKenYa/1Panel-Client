# 网站SSL证书模块索引

## 模块定位

网站SSL证书模块是Open1PanelApp的P1高价值扩展模块，提供网站级SSL证书的申请、更新、上传与绑定能力，是网站安全运行的重要保障。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **网站SSL证书** | 11 | ssl_v2.dart | 证书申请、更新、上传、下载、解析 |

## 后续规划方向

### 短期目标
- 完善网站证书信息展示
- 实现证书申请与更新流程

### 中期目标

### 长期目标

- 提供证书安全评估

## 与其他模块的关系

- **网站管理**: 证书与网站关联
- **监控管理**: 证书到期告警
- **日志管理**: 证书操作日志
- **系统设置**: 全局SSL配置
