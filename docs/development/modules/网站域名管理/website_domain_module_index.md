# 网站域名管理模块索引

## 模块定位

网站域名管理模块是Open1PanelApp的P1高价值扩展模块，提供网站域名的绑定与管理能力，是网站运行的基础配置模块。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **域名绑定** | 4 | website_v2.dart | 网站域名绑定、解绑与更新 |

## 后续规划方向

### 短期目标
- 完善域名绑定功能
- 补齐域名更新与SSL开关能力

### 中期目标
- 添加域名解析状态监控
- 支持批量域名管理

### 长期目标
- 实现域名监控告警
- 提供域名安全评估

## 与其他模块的关系

- **网站管理**: 域名与网站关联
- **SSL证书管理**: 域名证书绑定
- **监控管理**: 域名解析监控
- **日志管理**: 域名操作日志
