# Privacy Policy

Last updated: 2026-03-31

This policy applies to **1Panel Client** (the “App”).

## 1. Data Processing Principle

By default, the App is designed as a local management client and follows data minimization:

- We do **not** operate a cloud account system for this App.
- We do **not** collect personal information for centralized profiling.
- We do **not** sell or share personal information with advertising networks.

## 2. What Data May Be Processed Locally

To provide core functionality, the App may process the following data **on your device**:

- Server connection configuration (for example: server URL, API key)
- Local app settings (theme, language, preferences)
- Local diagnostic logs (when logging is enabled)
- Files explicitly exported by the user

## 3. Crash Logs, Telemetry, and Third-Party SDK

Current baseline:

- No mandatory remote telemetry upload is enabled by default.
- If future versions introduce crash collection, telemetry, or third-party SDK collection, we will:
  - disclose purpose, data categories, retention period, and processing basis;
  - provide withdrawal/opt-out mechanism when required;
  - request consent where required by applicable law.

## 4. Purposes of Processing

Data processing is limited to:

- establishing and maintaining your 1Panel server connection;
- showing runtime status and operation results;
- troubleshooting (through local logs and user-exported logs);
- improving product reliability and security.

## 5. Retention and Deletion

- Local logs are retained with automatic rotation/cleanup.
- Retention period is controlled by in-app logging configuration and policy.
- You can clear local logs and local app data from the app or device system settings.

## 6. User Rights

Subject to applicable law (including PRC PIPL where applicable), you may request:

- access to your local data in the App;
- correction of incorrect configuration data;
- deletion of local data;
- withdrawal of previously granted optional permissions.

## 7. Data Sharing

We do not share your local app data with third parties by default, except:

- when you explicitly export/share logs or files;
- when required by law.

## 8. Security Measures

- Sensitive local data is stored using platform security capabilities when applicable.
- We apply least-privilege and secure coding principles.
- We continuously improve logging desensitization and vulnerability handling.

## 9. Children

The App is intended for server administrators and technical users, not for children.

## 10. Changes to This Policy

We may update this policy. Material changes will be reflected in repository documentation and release notes.

## 11. Contact

For privacy/security concerns, please use:

- GitHub Issues: https://github.com/IsKenKenYa/1Panel-Client/issues
- Security reporting: see `SECURITY.md`
