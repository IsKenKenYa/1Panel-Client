# 隐私政策

最后更新：2026-03-31

本政策适用于 **1Panel Client**（以下简称“本应用”）。

## 1. 处理原则

本应用默认作为本地管理客户端运行，遵循最小必要原则：

- 默认不提供云端账号体系用于集中画像；
- 默认不进行个人信息集中采集；
- 默认不向广告网络出售或共享个人信息。

## 2. 本地可能处理的数据

为实现核心功能，本应用可能仅在你的设备本地处理以下数据：

- 服务器连接配置（例如：服务器地址、API Key）
- 本地应用设置（主题、语言、偏好）
- 本地诊断日志（启用日志时）
- 用户主动导出的文件

## 3. 崩溃日志、遥测与第三方 SDK

当前基线：

- 默认不启用强制远程遥测上传；
- 若后续版本引入崩溃收集、遥测或第三方 SDK 数据收集，将：
  - 明确告知处理目的、数据类型、保存期限、处理依据；
  - 在需要时提供撤回/关闭机制；
  - 依法在必要场景获取用户同意（含《个人信息保护法》要求）。

## 4. 处理目的

数据处理仅用于：

- 建立并维护你的 1Panel 服务器连接；
- 展示运行状态和操作结果；
- 通过本地日志与用户导出日志进行问题排查；
- 提升产品稳定性与安全性。

## 5. 保存期限与删除

- 本地日志采用自动轮转与清理策略；
- 保存周期由应用日志策略与配置决定；
- 你可通过应用或系统设置清理本地日志与本地数据。

## 6. 用户权利

在适用法律（含《个人信息保护法》）范围内，你享有：

- 访问本地数据的权利；
- 更正错误配置数据的权利；
- 删除本地数据的权利；
- 撤回已授予的可选权限的权利。

## 7. 对外提供

默认不向第三方提供你的本地数据，除非：

- 你主动执行导出/分享；
- 法律法规另有要求。

## 8. 安全措施

- 对敏感本地数据，优先使用平台安全存储能力；
- 采用最小权限与安全编码实践；
- 持续完善日志脱敏与漏洞治理流程。

## 9. 儿童保护

本应用面向服务器管理与技术用户，不面向儿童。

## 10. 政策更新

本政策可能更新。重大变更会在仓库文档与版本发布说明中体现。

## 11. 联系方式

隐私/安全问题请通过：

- GitHub Issues：https://github.com/IsKenKenYa/1Panel-Client/issues
- 安全披露：见 `SECURITY.md`
