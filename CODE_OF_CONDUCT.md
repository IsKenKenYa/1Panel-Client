# Code of Conduct

## Our Pledge

We are committed to making participation in this project respectful, constructive, and harassment-free for everyone.

## Our Standards

Examples of positive behavior:

- respectful communication
- constructive technical feedback
- empathy, patience, and good-faith collaboration
- focusing on user value, code quality, and maintainability

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, discrimination, or personal attacks
- abusive language, threats, or intimidation
- doxxing or sharing private information without permission
- disruptive trolling, spam, or bad-faith arguments
- repeated refusal to follow project rules after clarification

## Enforcement

Project maintainers are responsible for interpreting and enforcing this Code of Conduct.

They may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate these standards.

## Reporting

Please report unacceptable behavior through the project issue tracker:

https://github.com/IsKenKenYa/1Panel-Client/issues

If the report is security-sensitive, use the security reporting process instead of a public issue.

# 行为准则

## 我们的承诺

我们致力于让每一位参与者都能在尊重、建设性、无骚扰的环境中参与本项目。

## 行为标准

积极行为示例：

- 尊重他人，理性沟通
- 提供建设性的技术反馈
- 以同理心、耐心和善意协作
- 聚焦用户价值、代码质量与可维护性

## 不可接受的行为

以下行为不可接受：

- 骚扰、歧视或人身攻击
- 侮辱性语言、威胁或恐吓
- 未经允许公开他人隐私信息
- 恶意刷屏、故意捣乱或恶意争论
- 在明确提醒后仍持续拒绝遵守项目规则

## 执行与处理

项目维护者负责解释并执行本行为准则。

对于违反准则的评论、提交、Issue、Pull Request 或其他贡献，维护者有权删除、编辑、拒绝或限制其参与。

## 举报方式

如遇不可接受行为，请通过项目 Issue Tracker 报告：

https://github.com/IsKenKenYa/1Panel-Client/issues

如果问题涉及安全敏感内容，请改用安全报告流程，不要直接公开提交 Issue。
